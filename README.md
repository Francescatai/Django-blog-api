# Django-blog-api
## 技術


| 後端 |   前端 | 資料庫 |
| -------- | -------- | -------- |
| Django4     | Bootstrap4     | MySQL    |

 
 安裝django

```
pip3 install django
```

 開啟新項目

```
django-admin startproject [mysite]
cd [mysite]
python manage.py runserver
```
 創建程式
```
python manage.py startapp [blog]
```
